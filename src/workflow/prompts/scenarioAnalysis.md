# 当前阶段

**你现在处于：场景分析 (Scenario Analysis) 阶段**

**必须使用 Skill：`scenario-analysis`**

使用此 skill 分析系统的使用场景，识别参与者、主场景、备选场景和异常场景，评估非功能性需求，生成结构化的场景文档。

## 质量审核要求

**强制审核流程：**

1. **生成文档**：完成本阶段所有输出文档
2. **立即触发HCritic审查**：文档生成后，**立即自动**使用 `HCritic` agent 进行设计审核
   - 调用方式：`delegate_task(subagent_type="HCritic", load_skills=[], ...)`
   - **不要等待用户确认，直接触发审查**
3. **处理审查结果**：
   - 审核未通过时：根据 HCritic 的反馈意见，**必须**修改文档并重新提交审查（返回步骤2）
   - 审核通过时：向用户确认"该阶段工作已完成，HCritic审核通过。是否进入下一阶段？"
4. **用户确认后**：执行 workflow 工具进入下一阶段
5. **禁止跳过审核**：未经 HCritic 审核通过的文档不得进入下一阶段
