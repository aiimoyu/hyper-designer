# 阶段 1: 项目概览分析

## 重要说明

本阶段是 **project-to-skill** skill 的一部分。project-to-skill 是一个将项目代码库转化为可复用 skill 的完整工作流，包含4个阶段：
1. 项目概览分析（当前阶段）
2. 模块分析
3. 详细模块分析
4. SKILL.md 生成

在使用本阶段提示词时，请确保理解整个 skill 的上下文和目标。

## 任务

分析目标项目并生成4个核心文档。

## 前置条件

- 确认目标项目路径
- 读取 `references/Stage1-Overview.md` 获取完整方法论

## 输出

1. `Overview.md` — 项目概览
2. `Architecture.md` — 系统架构
3. `Guides.md` — 操作指南
4. `Principles.md` — 开发纲领（需与用户交互确认）

## 关键步骤

1. 并行探索（目录结构、技术栈、入口点、文档、配置）
2. 综合分析5个维度
3. 生成输出文件
4. 与用户确认开发纲领
5. 委派 subagent 验证架构完整性
